# TIL (Today I Learned)

毎日学んだことを簡潔にまとめて公開するリポジトリです。

## 目的

- 学習内容を定着させる
- 後で参照しやすくする
- 他の人と知識を共有する

## 方法

1. 学んだことをシンプルにまとめる。
2. 分かりやすいタイトルと内容で記録する。
3. 可能であれば、参考リンクやリソースを添付する。
